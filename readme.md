## 简介

`evidence_theory`是一个用于证据理论的Python包，提供了一系列工具和函数，帮助用户处理和应用证据理论。

## 安装

使用`pip`安装`evidence_theory`:

```bash
pip install evidence_theory
```

## 例子

见`./example/conflict_example.py`中实现了一个经典的证据冲突的融合。
